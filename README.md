
## Installation

- Install 

```python
  pip install python-telegram-bot firebase-admin pycryptodomex python-dotenv
```
- Setup .env file
```python
API_TOKEN=<your_api_token>
FIREBASE_CREDENTIALS=<firebase_credentials_path>
AES_KEY=<aes_key>
AES_IV=<aes_iv>
PRIVATE_KEY=<private_key_content>

```

- Run Script
```python
  python cryptobot.py
```
